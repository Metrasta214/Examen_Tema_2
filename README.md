# 🏠 Three.js FPS Demo – Colisiones con Octree

Este proyecto es una demo interactiva desarrollada con **Three.js**, que implementa un sistema de colisiones utilizando **Octree** y un controlador de jugador tipo **FPS (First Person Shooter)**.

---

## 🚀 Características

- 🎮 Movimiento tipo FPS (WASD + mouse)
- 🧱 Sistema de colisiones con Octree
- 🏠 Carga de modelos `.glb` (GLTFLoader)
- ⚙️ Física básica (gravedad, salto)
- 🔄 Sistema de respawn automático
- 🟢 Bounding Box para control de límites
- 🧪 Debug opcional con `OctreeHelper`

---

## 🕹️ Controles

| Acción         | Tecla             |
| -------------- | ----------------- |
| Moverse        | W / A / S / D     |
| Mirar          | Mouse             |
| Saltar         | SPACE             |
| Lanzar esfera  | Click             |
| Activar cámara | Click en pantalla |

---

## 🛠️ Tecnologías utilizadas

- [Three.js](https://threejs.org/)
- Octree (Three.js Addons)
- GLTFLoader
- JavaScript (ES Modules)
- HTML5 / CSS3
